# Privacy Policy

Last updated: 2025-11-30

This WhatsApp integration processes messages sent to the connected WhatsApp number for the sole purpose of delivering automated responses and related functionality.

## Data we receive
- WhatsApp message content and metadata delivered via the WhatsApp Cloud API (e.g., phone numbers, message IDs, timestamps).
- Webhook delivery logs (HTTP status, timestamps, diagnostics).

## How we use the data
- To receive, process, and respond to WhatsApp messages.
- To debug delivery issues and improve reliability.
